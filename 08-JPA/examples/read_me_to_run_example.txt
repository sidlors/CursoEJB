Complete the following steps before you open, deploy and run the JPAExample project in NetBeans.

1> In NetBeans, create a database EmployeeDB in JavaDB server. Provide the username and password as APP.
2>Keeping the connection open, open the Emp.sql file in NetBeans
3>Run the Emp.sql script file to create the Employee table in EmployeeDB and to populate it with some data.
4>Open the JPAExample project in NetBeans. Deploy in Oracle WLS and view the output in the browser window by providing the application URL.
